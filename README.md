# 📁 My Portfolio - Marcos Dinis

Welcome to my personal portfolio repository! This project is a simple web page that serves as a hub to showcase my first web development projects.

## 🚀 Overview

This is a basic portfolio website featuring:
- A homepage with my name (Marcos)
- Links to two previous projects
- Links to information pages (About and Contact)

The site is hosted on GitHub Pages and can be accessed at:
🔗 **[Marcos Portfolio](https://marcosdinis.github.io/html-portfolio/)**

## 📋 Featured Projects

### 1. 🎬 Movie Ranking
A simple HTML page listing the top three movies of all time, according to my personal preference. This project demonstrates:
- Basic HTML structure
- Ordered and unordered lists
- Text formatting

### 2. 🎂 Birthday Invite
An HTML page for a birthday invitation that includes:
- Event information
- Date and location
- Items to bring
- Location map

### 3. 📄 Additional Pages
- **About.html** - About me page
- **Contact.html** - Contact information page
